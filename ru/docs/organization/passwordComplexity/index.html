Сложность пароля

Casdoor поддерживает настройку вариантов сложности пароля для пользовательских паролей в каждой организации.

Поддерживаемые варианты сложности​

В настоящее время мы поддерживаем пять вариантов:

  • AtLeast6: Пароль должен содержать не менее шести символов.
  • AtLeast8: Пароль должен содержать не менее восьми символов.
  • Aa123: Пароль должен содержать как минимум одну заглавную букву, одну строчную букву и одну цифру.
  • SpecialChar: Пароль должен содержать как минимум один специальный символ.
  • NoRepeat: Пароль не должен содержать повторяющихся символов.

Если вы хотите использовать несколько вариантов, вы можете выбрать их на странице редактирования организации:

  1. Нажмите кнопку Редактировать на странице списка организаций.

  2. Затем выберите нужный вариант в столбце Варианты сложности пароля.

Проверка сложности пароля​

Мы поддерживаем проверку сложности пароля на следующих страницах:

  1. Страница регистрации.

  2. Страница восстановления пароля.

  3. Страница редактирования пользователя.
